# `editor.mini.surround`

Modify, add or delete surrounds (brackets, parens and more!)

## Hotkeys

- `gsa`: Add surround
- `gsd`: Delete surround
- `gsf`: Find to the right
- `gsF`: Find to the left
- `gsh`: Highlight surround
- `gsr`: Replace surround
